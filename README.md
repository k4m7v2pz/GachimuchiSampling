# GachimuchiSampling

本仓库收集并整理了用于制作鬼畜音乐的 **Gachimuchi（哲♂学兄贵）音频素材**，包括常见的 `.wav` 样本与 `.sf2` SoundFont 文件，可以在 Ardour 和 FL Studio 中使用。

这些素材来自经典 Gachimuchi 片段，包含高频率使用的台词、喘息、击打声等，便于快速、高效地进行音乐编排与混音处理。

---

## 📦 文件内容

- `.wav` 文件：音频样本（暂未将音量增大到最合适），通过原片标题创建的目录来分类。
- `.sf2` 文件：用于 MIDI 播放器和 DAW 的 SoundFont 音源

- Language of File Name（文件名语言）: Simplified Chinese（简体中文）
- Language of Voice（声音中讲的语言）: English（英语）
- Sample Rate（采样率）: 48000hz

---

## 🎵 适用场景

- 鬼畜 or 音 mad 制作
- DAW or Sampler（采样器）功能探索 / 声音设计
- 哲♂学爱好者的素材收藏

---

## 📁 使用方式

- Download Zip 或 在终端中使用 `git clone`
- 你可以将 `.sf2` 文件加载至支持 SoundFont 的软件中，如：

- **Ardour**：在内置插件 ACE Fluid Synth 中加载
- **FL Studio**：拖入通道机架即可创建新乐器
- **Polyphone**：`.sf2` 文件的编辑
